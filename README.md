# Проект BankWidget

## Описание:

BankWidget - виджет для шифрования номеров банковских карт и счетов, для сортировки по дате и исполению банковских операций клиента.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Rapunzel-Chan/homework.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
## Использование:

1. Функция get_mask_card_number маскирует номера банковской карты;
2. Функция get_mask_account маскирует номера банковского счета;
3. Функция filter_by_state возвращает список словарей, отсортированных по заданному параметру;
4. Функция sort_by_date возвращает список словарей, отсортированных по дате;
5. Функция get_mask_account_card маскирует номера как банковской карты, так и счета;
6. Функция get_data возвращает дату выпуска;
7. Функция filter_by_currency возвращает итератор, который поочередно выдает транзакции, 
где валюта операции соответствует заданной;
8. Функция transaction_descriptions возвращает описание каждой операции по очереди;
9. Функция card_number_generator выдает номера банковских карт в формате XXXX XXXX XXXX XXXX;
10. Декоратор log обеспечивает работу функции и ее результат как в файл, так и в консоль;
11. Функция create_and_read_transactions_file возвращает список словарей банковских операций, загруженных 
из внешнего хранилища;
12. Функция calculate_currency_exchange возвращает сумму трансакций в рублях с учетом курса валют.
13. Функция setup_logging создана для установки общих настроек логгеров, чтобы использовать их в других модулях;
14. Функции read_csv_transactions и read_xlsx_transactions созданы для чтения csv и excel файлов,
которые возвращают списки словарей после обработки.
15. Функции search_definite_transactions и search_category_transactions позволяют отфильтровать
банковские операции по конкретному описанию.
16. Функция main собирает всю функциональность виджета.

проверить работоспособность функций вне зависимости от внешних источников и

чтобы проверить правильность работы функций и обработки возникающих ошибок.
